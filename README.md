# 📊 Sistema de Vendas


Sistema simples para registro de vendas desenvolvido com HTML, CSS e
JavaScript.

------------------------------------------------------------------------

🚀 Funcionalidades

✔️ Adicionar vendas  
✔️ Calcular desconto automaticamente  
✔️ Exibir data e hora  
✔️ Remover vendas  
✔️ Limpar lista completa


------------------------------------------------------------------------

🛠️ Tecnologias

-   HTML5
-   CSS3
-   JavaScript

------------------------------------------------------------------------

📈 Melhorias futuras

-   Salvar dados no navegador
-   Filtro por vendedor
-   Responsividade mobile
-   Dashboard com gráficos
